Visi Kampanye

Kampanye ini berangkat dari meningkatnya aktivitas digital sehari-hari, khususnya di media sosial, yang sering kali dilakukan tanpa disadari dampak lingkungannya. Setiap unggahan, pencarian, streaming, penyimpanan data di cloud, hingga penggunaan layanan berbasis server jarak jauh membutuhkan energi yang besar, dan secara tidak langsung turut menyumbang jejak karbon digital. Ironisnya, sebagian besar konten yang dibagikan bersifat impulsif, berulang, atau kurang bermakna, hanya untuk mengejar validasi instan dan tren sesaat.

Melalui kampanye ini, kami ingin mengajak generasi muda, khususnya Gen Z sebagai pengguna internet terbesar, untuk lebih sadar, reflektif, dan bijak dalam beraktivitas di dunia digital, termasuk dalam penggunaan layanan cloud yang sering diakses tanpa kontrol dan kesadaran penuh.

Mengapa Website Ini Hemat Data?

Kami merancang website D'Grent dengan prinsip hemat data untuk mengurangi jejak karbon digital:

  • Menggunakan gambar vektor sederhana daripada foto berat
  • Mengoptimalkan kode untuk ukuran file yang minimal
  • Tidak menggunakan library eksternal yang besar
  • Mengurangi animasi yang tidak perlu
  • Menyimpan data lokal di browser pengguna

Dengan desain yang efisien, website ini hanya menggunakan sebagian kecil dari sumber daya server dibandingkan website konvensional.
